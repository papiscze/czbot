👋 Uvítací a odchodové zprávy

Automatické zprávy o příchodech a odchodech členů pro přátelský a informovaný server.

Uvítací zprávy jsou klíčové pro první dojem. Bot automaticky zveřejní zprávu s informacemi o novém členovi. Odchodové zprávy pomáhají sledovat dění na serveru.

Novinka: Nyní můžete nastavit samostatný kanál pro zprávy o odchodu. Pokud tak neučiníte, použije se stejný kanál jako pro příchody.

🛠️ Příkazy pro nastavení (POUZE MODERÁTOŘI)

Všechny funkce jsou spravovány pod příkazem /welcome.

1. Nastavení kanálů: /welcome set

/welcome set MOD

Nastaví cílový kanál pro uvítací zprávy a volitelně i pro odchodové zprávy.

Parametr Popis
welcome_channel Povinný: Kanál pro uvítací zprávy.
leave_channel Volitelný: Samostatný kanál pro odchodové zprávy.

Příklad s oddělenými kanály:

/welcome set welcome_channel:#prichody leave_channel:#odchody

Příklad s jedním kanálem:

/welcome set welcome_channel:#uvitaci-zpravy

2. Zobrazení nastavení: /welcome show

/welcome show MOD

Zobrazí, které kanály jsou aktuálně nastaveny pro uvítací a odchodové zprávy.

3. Odstranění nastavení: /welcome remove

/welcome remove MOD

Deaktivuje funkci a odstraní nastavení uvítacího i odchodového kanálu.

💬 Příklady zpráv

Uvítací zpráva:

Odchodová zpráva:
