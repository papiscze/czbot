📚 Průvodce používáním překladače (DeepL)

Rychlý a efektivní překlad zpráv a textu přímo na Discordu.

Tento modul využívá DeepL API pro poskytování vysoce kvalitních překladů. K dispozici máte:
1. Kontextové menu pro rychlý překlad zpráv.
2. Příkaz /prelozit pro překlad zadaného textu.
3. Automatický překlad kanálů pro mezinárodní komunity.

1. Rychlý překlad zprávy (Kontextové menu)

Doporučený způsob pro překlad zpráv, které již v chatu jsou.

Postup:

  1. Klikněte pravým tlačítkem na zprávu, kterou chcete přeložit.
  2. V menu vyberte Aplikace (Apps).
  3. Zvolte položku [BETA] Přeložit.
  4. Zadejte kód jazyka (např. CS) do cílového pole a odešlete.

Překlad se zobrazí pouze vám (ephemeralní odpověď).

2. Překlad textu pomocí příkazu /prelozit

Použijte, pokud chcete přeložit text, který píšete sami nebo který ještě není v chatu.

/prelozit UŽIVATEL

Parametr Popis
lang Kód cílového jazyka (např. DE pro Němčinu).
text Text k přeložení.

Příklad:

/prelozit lang:es text:Dobrý den, jak se máte?

3. Správa automatického překladu (POUZE MODERÁTOŘI)

Nastavení kanálů, kde se každá zpráva automaticky překládá do nastaveného jazyka.

/autopreklad setup MOD

Zapne automatický překlad pro daný kanál.

/autopreklad setup channel:#english-chat lang:en-gb

/autopreklad remove MOD

Zruší automatický překlad v daném kanálu.

/autopreklad remove channel:#english-chat

/autopreklad list MOD

Zobrazí seznam kanálů s nastaveným automatickým překladem.
