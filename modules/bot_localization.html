🌐 Lokalizace a jazyková nastavení

(Localization and Language Settings)

Tento modul umožňuje uživatelům a administrátorům serveru nastavit preferovaný jazyk, ve kterém s nimi bude bot komunikovat. Jazyk lze nastavit buď globálně pro celý server (nastavení má přednost pro všechny, kteří nemají nastavený osobní jazyk), nebo osobně pro jednotlivé uživatele.

⚙️ Správa jazyka

Všechny příkazy pro správu jazyka jsou sdruženy pod hlavním příkazem /lokalizace.

1. Nastavení jazyka pro celý server
Popis: Tento příkaz mohou používat pouze administrátoři s oprávněním `Spravovat server`. Nastavený jazyk bude použit pro všechny zprávy, které bot odesílá do veřejných kanálů, a jako výchozí jazyk pro všechny členy.
Příkaz: /lokalizace server
Detail: Nastaví výchozí jazyk bota pro daný server.
Parametry: kod_jazyka: Vyberte nový kód jazyka ze seznamu (např. Čeština (cs) nebo English (en)).
Příklad: /lokalizace server kod_jazyka:cs
Odpověď: ✅ Jazyk pro Server byl úspěšně nastaven na CS.
2. Nastavení osobního jazyka pro uživatele
Popis: Tento příkaz může použít kterýkoli uživatel k nastavení svého vlastního, osobního preferovaného jazyka. Osobní nastavení přepisuje nastavení serveru.
Příkaz: /lokalizace uzivatel
Detail: Nastaví váš osobní jazyk pro interakci s botem.
Parametry: kod_jazyka: Vyberte nový kód jazyka ze seznamu (např. Čeština (cs) nebo English (en)).
Příklad: /lokalizace uzivatel kod_jazyka:en
Odpověď: ✅ Jazyk pro Vás byl úspěšně nastaven na EN.

⚙️ Language Management

All language management commands are grouped under the main command /localization.

1. Set Server Language
Description: This command can only be used by administrators with the `Manage Server` permission. The set language will be used for all messages the bot sends to public channels and as the default language for all members.
Command: /localization server
Detail: Sets the bot's default language for the entire server.
Parameters: language_code (kod_jazyka): Select the new language code from the list (e.g., English (en) or Čeština (cs)).
Usage: /localization server language_code:en
Response: ✅ Language for Server has been successfully set to EN.
2. Set Personal User Language
Description: This command can be used by any user to set their own, personal preferred language. Personal settings override the server setting.
Command: /localization user
Detail: Sets your personal language for bot interactions.
Parameters: language_code (kod_jazyka): Select the new language code from the list (e.g., English (en) or Čeština (cs)).
Usage: /localization user language_code:cs
Response: ✅ Language for You has been successfully set to CS.
