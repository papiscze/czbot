🤫 Modul Zpovědi (Anonymní Zprávy)

Umožněte členům komunity posílat anonymní zprávy na server pomocí DM.

Zpovědi zajišťují maximální diskrétnost – zpráva se posílá přímo do DM botovi, který ji přepošle do nastaveného kanálu na serveru bez jakékoliv identifikace autora.

⚙️ Klíčová oprávnění bota

Pro správnou funkci je kritické oprávnění Vkládat odkazy (Embed Links).

  • Posílat zprávy: Pro odeslání zpovědi do cílového kanálu.
  • Vkládat odkazy: Bot posílá zpovědi jako formátované Embedy.

1. Administrace (Na Serveru)

Nastavení cílového kanálu je dostupné pouze pro Administrátory serveru.

/zpoved set ADMIN

Určí, který kanál bude sloužit pro zveřejňování anonymních zpovědí.

Parametr Popis
channel Povinný. Textový kanál, do kterého se zpovědi posílají.
/zpoved set channel:#tajemne-zpovedi

2. Odesílání Zpovědi (V DM s Botem)

Tento příkaz používají všichni uživatelé k odeslání zprávy. Musí být spuštěn v soukromé konverzaci s botem.

/send DM UŽIVATEL

Odešle anonymní zpověď na konkrétní server (pomocí ID).

Parametr Popis
server_id Povinný. Číselné ID serveru, kam se má zpověď doručit.
zpověď Povinný. Samotný text anonymní zprávy.
/send server_id:312537229799063555 zpověď:Miluji tento server!

🔑 Jak zjistit ID serveru?

  1. Jděte do Nastavení uživatele > Pokročilé a zapněte Režim vývojáře.
  2. Klikněte pravým tlačítkem myši na název serveru (v seznamu serverů) a zvolte Kopírovat ID.
  3. Zkopírované číslo použijte jako parametr server_id.

🛡️ Anonymita a Zabezpečení

Pro komunitu:

Zpráva v kanále obsahuje pouze náhodné Číslo zpovědi (např. #4567) a samotný text. Odesílatel je neznámý.

Pro administrátora (důvěrné):

Provozovatel bota má přístup k privátním logům, které zaznamenávají skutečné ID uživatele. To je určeno pouze pro řešení spamu, výhružek nebo ilegálního obsahu.
