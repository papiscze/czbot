🌟 XP Systém a Úrovně

Získejte XP za aktivitu, postupujte na vyšší úrovně a soutěžte v žebříčku!

XP systém odměňuje aktivní členy serveru za textovou aktivitu (s cooldownem proti spamu) a čas strávený v hlasových kanálech. Data jsou trvale uložena v databázi MySQL.

⚙️ Získávání XP

  • Text: XP za každou zprávu (cooldown: 60 sekund).
  • Hlas: XP za každou minutu v aktivním hlasovém kanále (nejsi ztlumený, 2+ uživatelé).
  • Progrese: Pro dosažení každé další úrovně je potřeba progresivně více XP.

Důležité oprávnění: Pro /xp config je potřeba oprávnění Spravovat server.

1. Správa XP Systému (pouze Admin)

/xp config ADMIN

Zapne/vypne XP systém a nastaví kanál pro oznámení o úrovních.

Parametr Popis
status True/False pro povolení/zakázání XP systému.
kanal Volitelný. Kanál pro oznámení o dosažení úrovně.
/xp config status:True kanal:#bot-commands

2. Uživatelské Příkazy (všichni členové)

/xp rank UŽIVATEL

Zobrazí tvůj nebo jiného uživatele rank, XP a progresi k další úrovni.

/xp rank member:@uzivatel

/xp top UŽIVATEL

Zobrazí žebříček top uživatelů serveru s nejvíce XP (podporuje stránkování).

/xp top

3. Moderační Příkazy (pouze MOD)

Tyto příkazy vyžadují oprávnění Spravovat zprávy.

/xp pridat_xp MOD

Ručně přidá zadané množství XP bodů uživateli.

Parametr Popis
uzivatel Uživatel, kterému XP přidáváte.
mnozstvi Počet XP bodů k přidání (kladné číslo).
/xp pridat_xp uzivatel:@JanNovak mnozstvi:500

/xp nastavit_uroven MOD

Ručně nastaví úroveň uživateli (XP se automaticky přepočítají).

Parametr Popis
uzivatel Uživatel, kterému úroveň nastavujete.
uroven Číslo úrovně (nemůže být záporné).
/xp nastavit_uroven uzivatel:@PetraDvorakova uroven:5
